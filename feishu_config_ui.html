📋
前置步骤：在飞书开放平台创建应用
只需配置一次，全项目共享
1
创建企业自建应用
访问 open.feishu.cn/app， 点击「创建企业自建应用」，填写应用名称和描述。
2
开启机器人能力
应用详情页 → 「功能」→「机器人」→ 点击启用。
3
添加权限范围
「权限管理」→「添加权限」→ 搜索并添加：
im:message  im:chat:readonly
4
发布应用并邀请机器人
点击「版本管理与发布」→「申请发布」（或测试版直接使用）。
然后在目标飞书群 → 「...」→「群机器人」→「添加机器人」→ 搜索你的应用名称。
5
获取 App ID 和 App Secret
应用详情页 →「凭证与基础信息」→ 复制 App ID 和 App Secret。
🔑
应用凭证
从飞书开放平台获取，仅存储在本地
将显示在飞书通知消息中
⚠️ 凭证仅保存在本地 .antigravity/feishu_config.json 中，不会上传
💬
选择目标群聊
消息将推送到此群，并从此群接收消息
在群设置中可查看，格式为 oc_ 开头
⚙️
通知与功能设置
自定义各功能的开关
🚀 打开项目时发送通知（功能1）
Antigravity 激活项目时，向飞书群发送就绪通知
✅ 对话完成后推送结果（功能3）
每次 Agent 完成响应后，将结果摘要发送到飞书群
📨 接收飞书消息作为对话输入（功能4）
需要运行 feishu_listener.py 守护进程
💅 使用消息卡片格式
消息卡片更美观，支持标题、Markdown 内容；否则使用纯文本
💾
保存配置
生成配置文件内容，在 Antigravity 终端中运行初始化命令
📡
启动消息监听器（功能4）
在 Antigravity 终端中运行，接收飞书群消息
# 前台运行（测试用） python feishu-bot/feishu_listener.py # 后台守护进程运行（推荐） python feishu-bot/feishu_listener.py --daemon # 查看运行状态 python feishu-bot/feishu_listener.py --status # 停止守护进程 python feishu-bot/feishu_listener.py --stop
监听器启动后，将在 .antigravity/feishu_messages.json 中写入收到的消息
